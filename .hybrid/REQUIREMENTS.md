# 需求规格

## 功能需求

### 1. 代理模式管理
- 支持TUN模式（透明代理）
- 支持系统代理（HTTP/SOCKS5）
- 支持TAP模式（二层代理）
- 支持混合模式
- 支持模式切换

### 2. 订阅管理
- 支持多个订阅源
- 支持订阅自动更新
- 支持节点筛选/分组
- 支持自动/手动排除无效节点
- 支持配置优先使用哪个订阅源下的节点
- 支持故障转移自动切换

### 3. 配置管理
- 支持交互式配置生成
- 支持黑白名单配置
- 支持配置导入导出
- 支持配置备份恢复

### 4. 服务管理
- 支持systemd服务管理
- 支持服务启停重启
- 支持日志查看

### 5. Web面板
- 集成mihomo-dashboard
- 支持Web界面管理

## 非功能需求

### 1. 性能需求
- 安装时间 < 1分钟
- 配置生成时间 < 10秒
- 内存占用 < 50MB
- CPU占用 < 5%

### 2. 兼容性需求
- 支持Ubuntu 18.04+
- 支持CentOS 7+
- 支持Debian 9+
- 支持WSL2

### 3. 安全需求
- 配置文件权限控制
- 敏感信息保护
- 安全的订阅更新

## 约束条件
- 纯bash实现，无额外依赖
- 单目录部署，不污染系统
- 支持GitHub分享

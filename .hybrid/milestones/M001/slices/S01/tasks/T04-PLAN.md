# T04 创建配置模板目录 任务计划

## 任务目标
创建配置模板目录和基础模板文件。

## 步骤分解

### 步骤1: 创建模板目录
```bash
mkdir -p templates
cd templates
```

### 步骤2: 创建基础配置模板
创建 base.yaml.template 文件，包含基本配置项。

### 步骤3: 创建TUN模式模板
创建 tun.yaml.template 文件，包含TUN模式配置。

### 步骤4: 创建系统代理模板
创建 system.yaml.template 文件，包含系统代理配置。

### 步骤5: 创建TAP模式模板
创建 tap.yaml.template 文件，包含TAP模式配置。

### 步骤6: 创建订阅模板
创建 providers.yaml.template 文件，包含订阅配置模板。

### 步骤7: 创建规则模板
创建 rules.yaml.template 文件，包含规则配置模板。

## 验收标准
1. 模板目录存在
2. 模板文件完整
3. 模板格式正确
